# מילון מונחים לתרגום Onetime Secret

## מינוח בסיסי

| English | עברית | Context |
|---------|-------|---------|
| secret (noun) | סוד | המושג המרכזי |
| passphrase | ביטוי סיסמה | הגנה על סודות |
| password | סיסמה | אימות חשבון |
| burn | מחיקה סופית | מחיקה לפני צפייה |
| link | קישור | כתובת URL |
| encrypt/encrypted | הצפנה/מוצפן | שיטת אבטחה |

## שיקולים מיוחדים

- העברית נכתבת מימין לשמאל (RTL)
- אין להשתמש בניקוד בעברית מודרנית
- הבחנה: "סיסמה" vs "ביטוי סיסמה"
- שימוש בגרשיים (״) ו

גרש (׳) לראשי תיבות
