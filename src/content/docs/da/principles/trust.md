# Gennemsigtighed og tillid

Vi mener, at tillid opnås gennem radikal gennemsigtighed om vores praksis, teknologier og forpligtelser.

## Vores gennemsigtighedsforpligtelser

- Open-source vores kerneteknologier
- Offentliggør klare, forståelige privatlivspolitikker
- Giv detaljerede sikkerheds- og infrastrukturrapporter
- Oprethold en løbende dialog med vores brugersamfund
- Undgå standard sporings- og marketingpraksis
- Hold dine data under direkte kontrol

## Hvorfor gennemsigtighed betyder noget

I en æra med komplekse digitale tjenester skiller vi os ud ved at være fuldstændig klar om, hvordan vi håndterer data, hvad vi gør, og hvorfor vi gør det. Vores tilgang centrerer om tre kerneprincipper:

1. Respekter privatliv: Dine data er dine, punktum
2. Fokus på pålidelighed: Værktøjet skal simpelthen fungere, når du har brug for det
3. Opbyg tillid gennem handling: Vi beviser vores engagement gennem, hvad vi undgår

## Vores privatlivs-først tilgang

Vi undgår bevidst almindelige industripraksisser, der kunne kompromittere privatlivet:

- Analyser, der sporer brugeradfærd
- Integrationsknapper til sociale medier
- Annoncenetværk
- Marketingautomatiseringsværktøjer
- Tredjepartssporingspixels

## Fremadrettet

Vi vil fortsætte med at prioritere gennemsigtighed og privatliv frem for markedsføring. Selvom dette betyder, at vi måske vokser langsommere end tjenester, der aggressivt markedsfører sig selv, mener vi, at det er det rigtige valg for at opbygge varig tillid hos vores brugere.
