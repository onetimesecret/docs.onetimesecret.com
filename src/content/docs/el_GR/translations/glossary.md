# Γλωσσάριο μετάφρασης Onetime Secret

## Βασική ορολογία

| English | Ελληνικά | Context |
|---------|----------|---------|
| secret (noun) | μυστικό | Κεντρική έννοια |
| passphrase | φράση πρόσβασης | Προστασία μυστικών |
| password | κωδικός πρόσβασης | Αυθεντικοποίηση λογαριασμού |
| burn | οριστική διαγραφή | Διαγραφή πριν την προβολή |
| link | σύνδεσμος | Διεύθυνση URL |
| encrypt/encrypted | κρυπτογράφηση/κρυπτογραφημένο | Μέθοδος ασφάλειας |

## Ειδικές παρατηρήσεις

- Μονοτονικό σύστημα (μονός τόνος από το 1982)
- Ερωτηματικό: ; (ημικόλονο)
- Διάκριση: "κωδικός πρόσβασης" vs "φράση πρόσβασης"
