# Översättningsordlista för Onetime Secret

## Grundläggande terminologi

| English | Svenska | Context |
|---------|---------|---------|
| secret (noun) | hemlighet | Centralt koncept |
| passphrase | lösenfras | Skydd av hemligheter |
| password | lösenord | Kontoautentisering |
| burn | radera permanent | Radering före visning |
| link | länk | URL-adress |
| encrypt/encrypted | kryptering/krypterad | Säkerhetsmetod |

## Särskilda överväganden

- Sammansatta ord skrivs ihop: lösenord, lösenfras
- Två genus (utrum och neutrum)
- Svenska bokstäver: å, ä, ö är väsentliga
- Skillnad: "lösenord" vs "lösenfras"
