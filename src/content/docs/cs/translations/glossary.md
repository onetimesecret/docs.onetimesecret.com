# Glosář překladu Onetime Secret

## Základní terminologie

| English | Čeština | Context |
|---------|---------|---------|
| secret (noun) | tajemství | Základní koncept |
| passphrase | přístupová fráze | Ochrana secrets |
| password | heslo | Autentizace účtu |
| burn | trvale smazat | Odstranění před zobrazením |
| link | odkaz | URL adresa |
| encrypt/encrypted | šifrovat/šifrovaný | Metoda bezpečnosti |

## Zvláštní úvahy

- Čeština má sedm pádů
- Diak

ritiku používat správně: á, č, ď, é, ě, í, ň, ó, ř, š, ť, ú, ů, ý, ž
- Rozlišení: "heslo" vs "přístupová fráze"
- Množná čísla: den/dny/dnů, hodina/hodiny/hodin
