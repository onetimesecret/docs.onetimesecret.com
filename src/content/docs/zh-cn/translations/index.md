# 翻译和本地化

Onetime Secret 文档有多种语言版本，为我们多样化的全球社区提供服务。我们将继续努力扩大我们的语言服务范围，并改进现有的翻译版本。

## 可用语言

Onetime Secret 文档目前有以下语言版本：

- [**English (en)**](/en/) - 标准和参考版本
- [**Български (bg)**](/bg/) - 保加利亚语
- [**Dansk (da)**](/da/) - 丹麦语
- [**Deutsch (de)**](/de/) - 德语
- [**Español (es)**](/es/) - 西班牙语
- [**Français (fr)**](/fr/) - 法语
- [**Italiano (it)**](/it/) - 意大利语
- [**日本語 (ja)**](/ja/) - 日语
- [**한국어 (ko)**](/ko/) - 韩语
- [**Māori (mi)**](/mi/) - 毛利语
- [**Nederlands (nl)**](/nl/) - 荷兰语
- [**Polski (pl)**](/pl/) - 波兰语
- [**Português (pt-BR)**](/pt-br/) - 葡萄牙语（巴西）
- [**Svenska (sv)**](/sv/) - 瑞典语
- [**Türkçe (tr)**](/tr/) - 土耳其语
- [**Українська (uk)**](/uk/) - 乌克兰语
- [**简体中文 (zh-CN)**](/zh-cn/) - 简体中文

您可以使用任何页面右上角的语言选择器在不同语言之间切换。

## 贡献翻译

我们欢迎大家为我们的翻译工作献计献策。如果您想帮助将 Onetime Secret 文档翻译成您的语言，请遵循以下指导原则：

### 翻译工作流程

1. **定位内容**：原始英文内容存储在 `/content/docs/en/` 目录中
2. **创建相应文件**：在您的语言目录中创建匹配文件（例如，`/content/docs/zh-cn/`）
3. **保持结构**：保持相同的目录结构和文件命名规则
4. **保留前置事项**：复制所有前台事务字段，只翻译值
5. **提交你的工作**：使用 [GitHub 拉取请求](https://github.com/onetimesecret/onetimesecret) 进行提交

### 翻译标准

要获得一致、高质量的翻译，请遵循我们的 [翻译风格指南](/zh-cn/translations/guide)，并参考我们的 [术语词汇表](/zh-cn/translations/glossary)，以获得关键术语的标准化翻译。

## 自动回退

当所选语言的内容不可用时，您将自动看到英文版本，并显示该内容尚未翻译。

如果您对翻译有疑问或需要帮助，请 [联系我们的团队](https://onetimesecret.com/contact) 或在我们的 [GitHub 仓库](https://github.com/onetimesecret/onetimesecret) 上打开一个问题。
