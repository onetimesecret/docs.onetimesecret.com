## Prompt

Provide translations for the noun ‘secret’ in the languages listed below, specifically for SaaS/security contexts. For each language, give:
- the primary technical noun for a discrete secret (e.g., API key),
- business-facing phrasing for confidentiality,
- UI example translations for ‘Create a new secret’, ‘Manage secrets’, ‘Confidential information’,
- notes on tone, common alternatives, and pitfalls.

Dutch, German, French, Spanish, Italian, Portuguese, Danish, Swedish, Norwegian, Polish, Japanese

en, de, nl, fr, es, uk, ko, ja, mi, bg, it, zh-cn, da, pl, pt-br, sv, tr
