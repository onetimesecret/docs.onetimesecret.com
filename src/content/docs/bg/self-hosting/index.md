Стартирайте собствена частна инстанция на Onetime Secret с пълен контрол върху вашите данни, сигурност и внедряване.

## Защо да хоствате самостоятелно?

Самостоятелното хостване на Onetime Secret ви дава:

- **Пълен контрол върху данните** - Всички тайни остават на вашата инфраструктура и мрежа
- **Персонализирани политики за сигурност** - Конфигурирайте автентикация, настройки за поверителност и контрол на достъпа
- **Съответствие** - Спазвайте нормативните изисквания за обработка на данни
- **Персонализиран брандинг** - Адаптирайте интерфейса за вашата организация

## Бързо стартиране

Изберете метода за внедряване, който най-добре отговаря на вашата среда:

### Docker (Препоръчително)
```bash
# Стартирайте Redis и Onetime Secret
docker run -p 6379:6379 -d redis:bookworm
docker run -p 3000:3000 -d \
  -e REDIS_URL=redis://host.docker.internal:6379/0 \
  -e SECRET="$(openssl rand -hex 32)" \
  onetimesecret/onetimesecret:v0.25.0
```

Достъп на `http://localhost:3000`.

### Ръчна инсталация
За продуктивни среди, изискващи персонализирани конфигурации.

Вижте нашето ръководство за [Инсталация и внедряване](./installation) за подробни стъпки.

## Какво е включено

Вашата самостоятелно хоствана инстанция включва:

- **Уеб интерфейс** - Пълнофункционален потребителски интерфейс за създаване и споделяне на тайни
- **REST API** - Програмен достъп за интеграции
- **Многоезична поддръжка** - Налична на повече от 12 езика
- **Персонализирани домейни** - Използвайте собствен домейн и брандинг

## Системни изисквания

**Препоръчителни:**
- 2+ процесорни ядра
- 2GB+ оперативна памет
- 10GB+ дисково пространство
- Redis за съхранение на сесии
- Node.js 22+ (за разработка)
