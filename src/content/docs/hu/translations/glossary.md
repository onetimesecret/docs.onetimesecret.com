# Onetime Secret fordítási szójegyzék

## Alapvető terminológia

| English | Magyar | Context |
|---------|--------|---------|
| secret (noun) | titok | Alapvető koncepció |
| passphrase | hozzáférési mondat | Titkok védelme |
| password | jelszó | Fiók hitelesítés |
| burn | végleges törlés | Törlés megtekintés előtt |
| link | hivatkozás | URL cím |
| encrypt/encrypted | titkosítás/titkosított | Biztonsági módszer |

## Különleges megfontolások

- Agglutináló nyelv, 18+ eset
- Magánhangzó-harmónia kötelező
- Ékezetek: á, é, í, ó, ö, ő, ú, ü, ű
- Nincs nyelvtani nem
- Megkülönböztetés: "jelszó" vs "hozzáférési mondat"
