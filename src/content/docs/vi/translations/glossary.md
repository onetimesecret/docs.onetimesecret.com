# Bảng thuật ngữ dịch Onetime Secret

## Thuật ngữ cơ bản

| English | Tiếng Việt | Context |
|---------|------------|---------|
| secret (noun) | bí mật | Khái niệm trung tâm |
| passphrase | cụm mật khẩu | Bảo vệ bí mật |
| password | mật khẩu | Xác thực tài khoản |
| burn | xóa vĩnh viễn | Xóa trước khi xem |
| link | liên kết | URL |
| encrypt/encrypted | mã hóa/đã mã hóa | Phương pháp bảo mật |

## Cân nhắc đặc biệt

- Ngôn ngữ phân tích, không có biến đổi hình thái
- 6 dấu thanh cần thiết để phân biệt ý nghĩa
- Không có giống ngữ pháp hay dạng số nhiều
- Phân biệt: "mật khẩu" vs "cụm mật khẩu"
