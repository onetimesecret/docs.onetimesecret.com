# 私たちのコミュニケーション・フィロソフィー

私たちは、ユーザーの時間と受信トレイを尊重し、ユーザーとのコミュニケーションに意図的に最小限のアプローチを取ります。

## 私たちのノー・メール・フィロソフィー

ほとんどのサービスは、あなたの受信トレイに殺到する：
- マーケティング・ニュースレター
- 製品の最新情報
- 利用統計
- 特集のお知らせ
- プロモーションオファー
- リエンゲージメントキャンペーン

私たちからお届けするメールは、シークレットリンクの配信、アカウント通知、または明示的にオプトインいただいた更新情報など、お客様が具体的にリクエストされたものだけです。オプトインいただいた場合でも、ご連絡はごくまれであり、ご登録いただいた目的に限定されます。受信トレイはお客様のものです。

## なぜこれが重要なのか

プライバシーツールは、マーケティングプラットフォームのように振る舞うべきではありません。コミュニケーションを最小限に保つことで：

- フィッシングやソーシャルエンジニアリングの攻撃対象を減らします — 私たちからの正規メールが少ないほど、不審なメールを見分けやすくなります
- エンゲージメント指標の最適化ではなく、信頼性の高いサービスの運営に集中できます
- メールアドレスを共有いただく際に寄せられる信頼を尊重します

## 私たちがエネルギーを注ぐこと

キャンペーンの作成ではなく、お客様にとって本当に重要なことに時間を費やしています：

- **コアの信頼性** — 必要なときにサービスが確実に動作すること
- **プライバシー保護** — 暗号化とデータ管理の強化
- **コンプライアンス** — 管轄区域および規制要件への対応
- **透明性** — 私たちの運営方法とその理由についてオープンであること
