# Russian (ru) Translation Notes

## Core Terminology

**Secret:** секрет (masculine) - Appropriate for professional contexts

**Password vs Passphrase:**
- Password: пароль - for account authentication
- Passphrase: кодовая фраза - for secret protection

**Burn:** окончательное удаление (permanent deletion)

## Key Points

- Cyrillic alphabet with 33 letters
- 6 grammatical cases
- 3 genders with proper agreement
- Plural forms: 1/2-4/5+ (день/дня/дней)
- Verb aspects: perfective vs imperfective
- Polite вы forms for professional tone
- Essential: use Ё where needed for clarity
