## Opções de Implantação

### Docker (Recomendado)

A maneira mais simples de implantar o Onetime Secret é usando Docker. Consulte o guia [Primeiros Passos](./getting-started) para instruções básicas do Docker.

### Manual

Para implantações manuais em ambientes de produção, você precisará configurar:

- Ruby 3.2+
- Redis 5+ ou Valkey
- Servidor web (nginx recomendado)
- Certificado SSL/TLS

Consulte a documentação completa no repositório do GitHub para etapas detalhadas.
