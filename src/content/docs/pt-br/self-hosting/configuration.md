## Configuração Básica

O Onetime Secret pode ser configurado através de:

1. **Variáveis de ambiente** - Recomendado para implantações Docker
2. **Arquivos de configuração** - Para implantações manuais

## Variáveis Principais

Consulte a página [Variáveis de Ambiente](./environment-variables) para uma lista completa de variáveis de configuração disponíveis.
