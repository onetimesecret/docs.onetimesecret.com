## Funcionalidades Principais

O Onetime Secret oferece uma solução direta para compartilhar informações sensíveis com segurança:

1. Acesso único: Visualize mensagens confidenciais compartilhadas uma vez e, em seguida, elas são excluídas permanentemente
2. Sem necessidade de registro: Use recursos básicos sem uma conta
3. Expiração personalizável: Controle a disponibilidade de mensagens confidenciais com exclusão automática
4. Interface intuitiva: Compartilhe mensagens confidenciais facilmente com nosso design amigável
5. Frase secreta opcional: Adicione uma camada extra de segurança às mensagens compartilhadas
6. Código aberto: Revise nosso código para maior transparência
